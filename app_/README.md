# APP - Frontend Next.js

Este é o frontend do Projeto 01 estruturado com Next.js (Pages Router).

## Scripts Disponíveis

Na pasta `app_`, você pode executar:

### `npm run dev`
Roda o servidor de desenvolvimento em `http://localhost:3001`.

### `npm run build`
Compila a aplicação para produção na pasta `.next`.

### `npm run start`
Inicia o servidor em modo de produção.
